# 🛠️ Artisan Pro Niger

Site vitrine moderne pour promouvoir des services d’artisans au Niger (plomberie, électricité, peinture, etc.).

---

## 🌐 Aperçu du projet

Ce projet est un site web **statique** développé avec :

- HTML5
- CSS3
- JavaScript (Vanilla)

Il permet de :
- Présenter des services professionnels
- Afficher une galerie de réalisations
- Permettre aux clients de contacter facilement
- Générer des prospects via WhatsApp et formulaire

---

## 🚀 Fonctionnalités

✔ Page d’accueil moderne et attractive  
✔ Navbar responsive (mobile + desktop)  
✔ Section services claire et structurée  
✔ Galerie d’images dynamique  
✔ Formulaire de contact (via Formspree ou email service)  
✔ Bouton WhatsApp flottant  
✔ Animations au scroll  
✔ Design responsive (mobile first)

---

## 🧱 Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
- Formspree (optionnel pour formulaire sans backend)
