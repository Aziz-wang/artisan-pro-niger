🛠️ Artisan Pro Niger

Site vitrine moderne pour promouvoir des services d’artisans (plomberie, électricité, peinture, etc.) au Niger.

---

🌐 Aperçu du projet

Ce projet est un site web statique (HTML, CSS, JavaScript) conçu pour :

- Présenter des services d’artisans
- Montrer des réalisations (galerie)
- Permettre aux clients de contacter facilement
- Générer des prospects via formulaire et WhatsApp

---

🚀 Fonctionnalités

✔ Page d’accueil attractive
✔ Navbar responsive (mobile + desktop)
✔ Section services
✔ Galerie d’images
✔ Formulaire de contact (sans backend via Formspree)
✔ Bouton WhatsApp flottant
✔ Animations au scroll
✔ Design responsive

---

🧱 Technologies utilisées

- HTML5
- CSS3
- JavaScript (vanilla)
- Formspree (pour les formulaires sans backend)

⚙️ Installation (local)

Clone ou télécharge le projet :

git clone https://github.com/ton-username/artisan-site.git

Lance le serveur local :

python -m http.server 8000

Ouvre dans le navigateur :

http://127.0.0.1:8000

---

📩 Formulaire de contact

Le formulaire utilise Formspree.

👉 Pour l’activer :

1. Créer un compte sur Formspree
2. Créer un formulaire
3. Remplacer dans "contact.html" :

action="https://formspree.io/f/TON_ID"
